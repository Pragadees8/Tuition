# Tutor CRM

## Overview

Tutor CRM is a backend application developed using Java and Spring Boot to manage online tuition operations. The system supports student management, teacher management, course materials, homework tracking, attendance monitoring, fee management, and real-time notifications.

## Features

### Authentication & Security
- JWT Authentication
- Role-Based Authorization
- Secure REST APIs
- Spring Security Integration

### User Management
- Super Admin Management
- Admin Management
- Sub Admin Management
- Teacher Management
- Student Management

### Academic Management
- Subject Management
- Enrollment Management
- Homework Management
- Homework Submission Tracking
- Attendance Management

### Learning Resources
- Study Material Management
- Video Management

### Communication
- Real-Time Notifications using WebSocket
- Feedback Management
- Proposal Management

### Finance
- Student Fee Management

## Technologies Used

- Java 17

- RESTful API Architecture
